📊 GHOST EATING SCORE FORMULA:

Formula: points = 200 * Math.pow(2, this.ghostsEaten - 1)

Examples:
  ├─ 1st ghost eaten: 200 * 2^(1-1) = 200 * 1 = 200 points
  ├─ 2nd ghost eaten: 200 * 2^(2-1) = 200 * 2 = 400 points
  ├─ 3rd ghost eaten: 200 * 2^(3-1) = 200 * 4 = 800 points
  ├─ 4th ghost eaten: 200 * 2^(4-1) = 200 * 8 = 1600 points
  └─ 5th ghost eaten: 200 * 2^(5-1) = 200 * 16 = 3200 points

Reset Condition:
  Power pellet collection → this.ghostsEaten = 0 → Next ghost = 200 points

📊 POWER-UP SYSTEM TIMELINE:

1. ACTIVATION (checkDots detects POWER cell):
   └─ frightTimer = getLevelFrightTime()
   └─ All ghosts (except eaten) → GM_FRIGHTENED mode
   └─ Ghost directions reversed

2. ACTIVE PHASE (each update frame):
   ├─ frightTimer-- (decrements each frame)
   ├─ updateGhostMode() checks: if (this.frightTimer > 0) { return; }
   └─ While frightTimer > 0:
      └─ Ghosts move randomly (no chase)
      └─ Can be eaten (eaten → GM_EATEN, return home)

3. TERMINATION:
   Option A - Timer expires:
   ├─ frightTimer <= 0
   ├─ For each ghost in GM_FRIGHTENED:
   │  ├─ ghost.mode = this.globalMode (previous mode)
   │  ├─ ghost.speed = getSpeed('ghost', g)
   │  └─ ghost._lastDecisionTile = -1
   └─ this.ghostsEaten = 0 (reset multiplier)
   
   Option B - Eaten ghost returns home:
   ├─ Ghost reaches (col ≈ 14, row ≈ 12)
   ├─ ghost.inHouse = true
   ├─ ghost.mode = (frightTimer > 0) ? GM_FRIGHTENED : globalMode
   └─ Ghost stays home until respawned
