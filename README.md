# First Frame Studios

> Forged in Play

A multi-genre game development studio. This monorepo contains all games, shared assets, and studio infrastructure.

## Structure

| Directory | Purpose |
|-----------|---------|
| `.squad/` | Studio team, skills, decisions, identity |
| `games/` | One folder per game project |
| `shared/` | Assets and code shared between games |
| `docs/` | Studio documentation and guides |
| `.github/` | CI/CD workflows and GitHub configuration |
