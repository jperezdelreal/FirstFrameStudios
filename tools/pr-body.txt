## Sprint A Wave 1: Quick Wins 🚀

This PR delivers the first 4 tools from the prioritized tools list — all quick wins that can ship immediately.

---

## Tools Delivered

### 1. Branch Validation CI Action ✅
**File:** `.github/workflows/branch-validation.yml`  
**Impact:** 🔴 CRITICAL — Prevents Tarkin-class dead branch failures  
**Effort:** 30 minutes

**What it does:**
- Automatically checks that all PRs target `main` branch
- Blocks merge if PR targets wrong branch
- Prevents coordination disasters

### 2. PR Body Validator ⚠️
**File:** `.github/workflows/pr-body-check.yml`  
**Impact:** 🟡 HIGH — Ensures PRs are traceable  
**Effort:** 1 hour

**What it does:**
- Checks for `Closes #N` pattern in PR body
- Posts friendly reminder comment if missing
- Does NOT block merge (warning only)

### 3. Autoload Dependency Analyzer 🔧
**File:** `tools/check-autoloads.py`  
**Impact:** 🔴 CRITICAL — Prevents M1+M2 Blocker #5  
**Effort:** 2-3 hours

**What it does:**
- Validates autoload order in `project.godot`
- Checks all autoload files exist
- Detects dependency violations (e.g., SceneManager loading before RoundManager)
- Suggests correct order if validation fails

**Run it:**
```bash
python tools/check-autoloads.py
```

### 4. Signal Wiring Validator 🔌
**File:** `tools/check-signals.py`  
**Impact:** 🟡 HIGH — Prevents M1+M2 Blocker #2  
**Effort:** 3-4 hours

**What it does:**
- Scans all .gd files for signal definitions, emissions, and connections
- Reports orphaned signals (emitted but never connected, or vice versa)
- Outputs detailed wiring matrix
- Detects silent integration failures

**Run it:**
```bash
python tools/check-signals.py
```
