## Key Specification Parameters

### Outlines
- **Thickness:** 0.008 (4× increase from 0.002) — ~2–3 pixels at 512×512 PNG
- **Kael Color:** (0.35, 0.15, 0.05) — burnt sienna, warm tint
- **Rhena Color:** (0.08, 0.12, 0.20) — dark navy, cool tint
- **Rationale:** Tinted outlines (not pure black) harmonize with character palette

### Shading
- **Steps:** 2 (down from 3) — hard shadow/lit split at 0.5 diffuse threshold
- **Interpolation:** CONSTANT (hard edge, no smoothing)
- **Shadow/Base Colors:** Locked to existing character presets

### Lighting
- **Key Light:** SUN, energy 3.0, rotation (50°, 10°, 30°) XYZ
- **Fill Light:** SUN, energy 1.5, rotation (60°, -20°, -30°) XYZ
- **World Ambient:** (0.15, 0.15, 0.18), strength 0.5

### Export
- **Format:** PNG RGBA 8-bit
- **Renderer:** BLENDER_EEVEE
- **Post-processing:** Optional bloom (test without first)

## Decision Summary

**Problem:** Outline thickness (0.002) is invisible at 512×512 PNG export; reads as muddy. 3-step shading too soft for fighting game punch.

**Solution:** Adopt Guilty Gear Xrd cel-shade approach:
1. Bold, readable outlines (0.008)
2. Hard-edged 2-step shadow/lit banding
3. Dramatic directional lighting
4. Color-restrained design (tinted outlines, no radical hue shifts)

**Impact:**
- Visual: Immediate character recognizability at all scales; arcade-game polish
- Technical: Minor code changes, no render time increase
- Schedule: 3–5 hours total (Chewie implementation + testing)
