## Key Lesson Learned

**Multi-agent integration seams are high-risk.** When different agents own emitter (hitbox.gd) → signal → consumer (fighter_base.gd), the full chain must be tested end-to-end before PR merge. Signal signatures and scene node hierarchies are the most common mismatch points.

**Team process improvement:** Cross-module PRs require a "integration checklist" — verify all call sites, all signal receivers, and run a live integration test.
