## Heartbeat Protocol

**Agent Heartbeat Signal (JSON):**
```json
{
  "agent_id": "solo",
  "timestamp": "2026-03-11T16:42:00Z",
  "status": "active|idle|stalled",
  "last_task": "PR review Flora #20",
  "uptime_seconds": 3600,
  "version": "1.0"
}
```

**Heartbeat Reader Response:**
```json
{
  "agents": [
    {
      "id": "solo",
      "status": "active",
      "last_seen": "2026-03-11T16:42:05Z",
      "health": "healthy|degraded|offline"
    }
  ],
  "timestamp": "2026-03-11T16:42:10Z"
}
```
