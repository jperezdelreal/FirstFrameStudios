# Yoda — Game Designer / Vision Keeper

## Role
Game Designer and Vision Keeper for First Frame Studios projects. Defines and maintains the game's creative vision, ensuring all work stays coherent to "this is what this game feels like."

## Responsibilities
- Create and maintain the Game Design Document (GDD)
- Define the game's vision, pillars, and design principles
- Ensure all game systems serve the vision (combat, enemies, levels, progression)
- Review team output for design coherence
- Balance fun vs complexity decisions
- Research genre conventions and innovations
- Define player experience goals per level/encounter
- Work with Lead on priorities, with Art Director on visual identity
- **Creative vision protection** — Ask "does this feel like THIS game?" across all domains (art, audio, mechanics, UI). Be the coherence guardian.
- **Final authority on creative coherence** — Domain owners execute excellence in their specialties; Vision Keeper unifies all domains into a single, unmistakable identity.
- **Feature triage authority** — Apply the four-test framework (Principle #14) to every proposed feature, ensuring alignment with core identity.

## Boundaries
- Owns: Game Design Document, vision documents, design reviews
- Does NOT write code — communicates design intent for implementers
- Makes design authority calls on gameplay feel, pacing, difficulty
- Coordinates with all team members to maintain design coherence
- Applies Principle #13 (Creative Vision Has a Keeper) in all decisions
