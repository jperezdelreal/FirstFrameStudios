# About First Frame Studios

First Frame Studios is an AI-powered game development studio founded by [Joaquín Pérez del Real](https://github.com/jperezdelreal).

## The Model

We use [Squad](https://github.com/bradygaster/squad) to coordinate a team of 15 specialized AI agents. Each agent owns a domain — game design, engine development, art direction, audio, QA, production, tooling — and works autonomously on GitHub issues, filing PRs and iterating through the same development cycle a human team would follow.

The difference: our agents work in parallel, around the clock, and never forget a decision made three sprints ago.

## The Philosophy

- **Ship games, not demos.** Every project targets a playable build within the first sprint.
- **Player feel is sacred.** Architecture serves the player's experience, not the codebase's elegance.
- **Research is ammunition.** We study the greats in every genre before writing gameplay code.
- **Every specialist owns their domain.** When domains blur, quality drops.

## The Journey

We started with **Ashfall** — a 1v1 fighting game in Godot 4. We learned that AI teams build infrastructure fast but fighting games need human feel-tuning.

Then **firstPunch** — a beat 'em up with a 1,931 LOC custom engine in vanilla JS. We proved AI agents can build real, working game systems from scratch.

Now we're building **ComeRosquillas** (an arcade web game) and **FLORA** (a cozy roguelite in PixiJS), while our autonomous loop keeps the studio running 24/7.
