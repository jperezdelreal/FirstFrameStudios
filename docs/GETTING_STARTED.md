# Getting Started with First Frame Studios

## Prerequisites
- Git
- Godot 4.x (for Ashfall and future projects)
- A modern browser (for First Punch)

## Repository Layout
This is a monorepo. Each game lives in `games/{game-name}/`.
Studio knowledge lives in `.squad/`.

## Running First Punch
```bash
cd games/first-punch
# Open index.html in a browser, or:
npx serve .
```

## Working on Ashfall
```bash
cd games/ashfall
# Open with Godot 4 editor
```

## Team
Read `.squad/team.md` for the roster and `.squad/identity/` for studio principles.
